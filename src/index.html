Утилиты КЖ

1. Расчет плoщади арматуры по кол-ву стержней разных диаметров

0см2

Шаблон: nN1 d D1 + N2 d D2 + ... + Ni d Di

где Ni - кол-во стержней в диапазоне [1..12], Di - диаметр арматуры по сортаменту

2. Расчет площади арматуры на метр погонный по шагу стержней разных диаметров

0см2/м.пог.

Шаблон: d D1 s S1 + d D2 s S2 + ... + d Di s Si

где Di - диаметр арматуры по сортаменту, Si - шаг стержней в мм из множества [1000, 400, 350, 300, 250, 200, 150, 100, 50]
